# CommunityBot

Ein moderner Discord-Bot mit discord.py 2.x, Slash Commands, SQLite, Cog-System, Musik, Giveaways, Tickets, Moderation, Levelsystem, Willkommen/Verabschiedung, Dashboard und Logging.

## Schnellstart

1. Python 3.10+ installieren
2. Abhängigkeiten installieren: `pip install -r requirements.txt`
3. `.env` aus `.env.example` kopieren und anpassen
4. Bot starten: `python bot.py`
5. Dashboard starten: `python dashboard/web.py`
